Crear el archivo `frontend/src/lib/descriptor.js` con dos funciones exportadas puras.

**`detectSingleChain(s)`**
- Recibe string descriptor.
- Devuelve `true` si el descriptor contiene al menos un `/N/*` (donde N es entero no negativo) que NO está ya dentro de `<a;b>`.
- Regex: `/\/(\d+)\/\*/` — si hay match, el descriptor es single-chain.
- El patrón `<0;1>/*` no contiene `/0/*` literal, así que no hay colisión con multipath ya correcto.
- IMPORTANTE: `/0h/*` y `/0'/*` (derivación hardened) NO deben disparar la detección — la regex `/\/(\d+)\/\*/` ya los excluye porque `h` o `'` interrumpen el match.
- Si el string no es string o está vacío, devuelve `false`.

**`convertSingleChainToMultipath(s)`**
- Recibe string descriptor.
- Usa la **regla par-impar sibling** (alineada con Sparrow/BIP44/BIP389):
  - Helper interno: `function pairFor(n) { const base = Math.floor(n / 2) * 2; return [base, base + 1]; }`
  - `/0/*` → `<0;1>/*` (base=0)
  - `/1/*` → `<0;1>/*` (base=0) — CLAVE: NO `<1;2>/*`
  - `/2/*` → `<2;3>/*` (base=2, cubre Liana recovery)
  - `/3/*` → `<2;3>/*` (base=2)
  - Casos N≥4 escalan con la misma fórmula
- Usa `String.prototype.replace` con regex global `/\/(\d+)\/\*/g` y función de reemplazo:
  ```js
  function pairFor(n) {
    const base = Math.floor(n / 2) * 2;
    return [base, base + 1];
  }
  // en replace:
  (_, n) => {
    const [a, b] = pairFor(parseInt(n, 10));
    return `/<${a};${b}>/*`;
  }
  ```
- Después del reemplazo, elimina el checksum BIP-380 trailing: strip `/#[a-z0-9]{8}$/i` del resultado (con trim de espacios opcionales antes del `#`).
- Devuelve el descriptor transformado.

**Caso mixto (descriptor con mezcla de cosigners):**
- Si un cosigner ya usa `<0;1>/*` y otro usa `/0/*`, la regex global solo matchea el que tiene `/0/*` (el patrón `/<0;1>/*` no matchea `/\d+/*`). La función convierte SOLO los cosigners que no son multipath — comportamiento correcto sin lógica especial.

Seguir el estilo de `xpub.js`: comentario de cabecera explicando el propósito, JSDoc mínimo para cada función.

Ejemplo esperado:
- Input:  `wsh(sortedmulti(2,[fp/48h/0h/0h/2h]xpub.../0/*,[fp/48h/0h/0h/2h]xpub.../0/*))#abc12345`
- Output: `wsh(sortedmulti(2,[fp/48h/0h/0h/2h]xpub.../<0;1>/*,[fp/48h/0h/0h/2h]xpub.../<0;1>/*))` (sin checksum)

Ejemplo con N=2 (Liana recovery chain):
- Input:  `wsh(or_d(pk([fp]xpub.../2/*),and_v(…)))#xxxxxxxx`
- Output: `wsh(or_d(pk([fp]xpub.../<2;3>/*),and_v(…)))`

Ejemplo con /1/* (caso crítico — change-only descriptor Nunchuk):
- Input:  `wpkh([fp]xpub.../1/*)#deadbeef`
- Output: `wpkh([fp]xpub.../<0;1>/*)`  ← NO `<1;2>/*`

No exportar nada más. No importar dependencias externas.
